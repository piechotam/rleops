# Filter an rle vector by another logical rle vector

Filter an rle vector by another logical rle vector

## Usage

``` r
rle_filter(rle_data, rle_mask)
```

## Arguments

- rle_data:

  An R rle list object with data to filter

- rle_mask:

  An R rle list object with logical mask

## Value

A filtered rle list object
